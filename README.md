# SMG Repository
This is the repository for Sleep Management Group's website (v2.0).
legacy: http://sleepmanagement.ca
demo: http://smgdemo.xyz

API contains nodeJS/EXPRESS backend. WEB contains ReactJS/Redux/MaterialUI frontend.

This is a work in process, there may be todos and commented code blocks.

# To Run:

with Docker: Run "docker-compose up --build" to build image and mount to container

else: Otherwise, each service can be started with commands found in the "package.json" file in each directory
